SpatialTools - Open source spatial processing

SpatialTools are a collection of simple command line programs written in C and available under the GNU General Public Licence (GPL). The programs allow to achieve some very basic tasks that are difficult to achieve using common GIS software.

Currently, all programs in the SpatialTools package are related to raster processing. The naming conventions of GRASS GIS have been used.

SpatialTools contains currently the following programs:

  • r.clusters.stat Computes statistics for different clusters
  • r.comparison.plot Allows comparing two raster images
  • r.downscale Makes an areal interpolation using dasymmetric mapping.
  • r.fdim.boxcount Estimates the fractal dimension using the boxcounting method.
  • r.groupcells Groups several cells of a raster together. This is a special case of resampling, or a sort of spatial group by.
  • r.lacunarity Computes the lacunarity measure for a raster image
  • r.percolation Computes the spatial clusters in a raster map, based on percolation.
  • r.potential Computes a potential surface based on an input raster and a spatial variogram.
  • r.to.geoeas Writes a raster file into a GeoEAS text file.

Download

All the source code is written in C language and should compile using a more or less modern GCC compiler. The code has been written using XCode on Mac OS X; the XCode projects are contained in the download. Currently, no make files are present; if somebody is preparing compilation using the make system, I would be happy to include those in this distribution.
All the programs make use of the GDAL library that has to be downloaded separately (version 1.5 should be fine). On Mac OS X, you can use the precompiled frameworks from kyngchaos.com.
