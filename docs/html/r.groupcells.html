List of programs

r.groupcells

Name

r.groupcells -- groups several cells of a raster together. This is a special case of resampling, or a sort of spatial group by.

Keywords

raster, resampling

Synopsis

r.groupcells
[-h] -r value [-f format] [-s statistic] input_raster output_raster

Description

The following options are available:

-h
Shows this usage note.

-r value
The number of cells to group into one cell.

-s statistic
Statistic to use for grouping the cells together. Following options are available: sum, min, max, mean. Default is mean.

-f format
Format for the output raster file. Default is GTiff. The following formats are supported:

GTiffGeoTIFF
HFAErdas Imagine (.img)
AAIGridArc/Info ASCII Grid
PNGPortable Network Graphics
JPEGJPEG image
GIFGraphics Interchange Format
PCIDSKPCIDSK Database File
PCRasterPCRaster Raster File
GMTGMT NetCDF Grid Format
JPEG2000JPEG-2000
RSTIdrisi Raster A.1
ENVIENVI .hdr Labelled
