r.to.geoeas

Name

r.to.geoeas -- Writes a raster file into a GeoEAS text file.

Keywords

raster, GeoEAS

Synopsis

r.to.geoeas
[-h] [-n value] input_raster output_eas

Description

The following options are available:

-h
Shows this usage note.

-b band
The band to convert. Default is 1.

-n value
No data value which is not written to the output file.

input_raster
Raster file to convert.

output_eas
The output GeoEAS file.
